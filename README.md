## general references

### build
`go build -o reference`

### use
`./reference -i input.ref -o reference.md -p http://127.0.0.1:9999`

- -i : input file path
- -o : output file path
- -p : proxy

### example
- input file:
```
https://time.geekbang.org/
https://www.google.com.hk/
```

- output file:
- [ ] [极客时间-轻松学习，高效学习-极客邦](https://time.geekbang.org/)
- [ ] [Google](https://www.google.com.hk/)
